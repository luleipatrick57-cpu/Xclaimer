# 📥 How to Download & Run Xclaimer Bot Pro

## Quick Download (2 minutes)

### Method 1: Download as ZIP (Easiest)
1. Go to: https://github.com/luleipatrick57-cpu/Xclaimer
2. Click **Code** (green button) → **Download ZIP**
3. Extract the ZIP file to any folder
4. Double-click **index.html** to open in browser
5. Done! ✅

### Method 2: Clone with Git
```bash
git clone https://github.com/luleipatrick57-cpu/Xclaimer.git
cd Xclaimer
# Open index.html in your browser
```

### Method 3: Individual Files
Save these 3 files in the same folder:
- Download from: https://github.com/luleipatrick57-cpu/Xclaimer/blob/main/index.html
- Download from: https://github.com/luleipatrick57-cpu/Xclaimer/blob/main/style.css
- Download from: https://github.com/luleipatrick57-cpu/Xclaimer/blob/main/app.js

Then open `index.html` in your browser.

## Step-by-Step Setup

### Step 1: Download Files
- Use any method above to get the 3 files:
  - `index.html`
  - `style.css`
  - `app.js`

### Step 2: Create a Folder
- Create a new folder on your computer (e.g., "Xclaimer")
- Place all 3 files in this folder

### Step 3: Open the Bot
- **Windows**: Right-click `index.html` → Open with → Choose browser
- **Mac**: Double-click `index.html` (opens in default browser)
- **Linux**: Right-click `index.html` → Open with → Your browser

✅ You should now see the bot interface!

## Get Your Deriv API Token

1. Visit: https://app.deriv.com
2. Login or create account
3. Go to **Settings** → **API tokens**
4. Click **Create New Token**
5. Add these scopes:
   - ✅ Trade
   - ✅ Read
   - ✅ Admin
6. Set expiration (1 year or custom)
7. Click **Create**
8. Copy the long token string

---

## First Run

1. **Paste API Token**
   - Copy your token from Deriv
   - Paste in "Enter Deriv API Token" field
   
2. **Click Connect** 🔗
   - Wait for "Authorization successful"
   - Your balance will display

3. **Configure Settings**
   - Market: R_25 (recommended for testing)
   - Stake: $0.35 (minimum $0.01)
   - Take Profit: $20
   - Stop Loss: $15

4. **Click Start** ▶
   - Bot starts analyzing prices
   - Wait for signal generation
   - Trades execute automatically

5. **Monitor & Stop**
   - Watch Trade Log for updates
   - Click Stop ⏹ to pause

## Troubleshooting

**Q: It won't open**
- Make sure all 3 files are in the same folder
- Try a different browser
- Check internet connection

**Q: API connection fails**
- Verify your token is correct (copy-paste again)
- Create a new token
- Check internet connection

**Q: Bot not trading**
- Check Trade Log for errors
- Ensure account has minimum balance ($0.01 + stake)
- Try demo account first

**Q: File permissions error**
- Save files to Documents or Desktop
- Try different folder location
